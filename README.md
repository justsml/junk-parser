# junk-parser

### Currently Parses

* handles 2-line csv, extra line-break
* handles 2-line quoted csv, xtra line-break
* handles 2 extra line-breaks
* handles 1 row on 4 lines, w/ "empty" line
* handles 2 row, 4 lines quoted csv w/ trailing delimiter
